## Current Projects
### Hallucination Detection in Domain-Tuned LLMs
Working on developing an automated system to detect hallucinations in domain-specific LLMs for material science. This project involves leveraging stochastic response generation, named entity recognition (NER), and semantic similarity checks to ensure factual accuracy, funded by the Department of Defense (DoD).

### Brain Tumor Detection in MRI Images
Developed CNN and ResNet50V2 models for brain tumor detection using transfer learning and data augmentation techniques. This project enhanced model accuracy by 15% and involved high-performance computing on Nautilus.
